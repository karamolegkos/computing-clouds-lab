# Εκτέλεση του Docker Hello World 🐳

Παρακάτω γίνεται βήμα προς βήμα εξήγηση του πώς να εκτελέσετε το κλασικό "**Hello World**" container του Docker, ώστε να επιβεβαιώσετε ότι το Docker είναι σωστά εγκατεστημένο και λειτουργεί στο σύστημά σας.

## Προαπαιτούμενα
- Docker Engine

Μπορείτε να εξετάσετε την εγκατάσταση του Docker σας τρέχοντας το παρακάτω:
```shell
docker --version
# Πρέπει να λάβεται κάτι τέτοιο:
Docker version 27.0.1, build abc1234
```

## Εκτέλεση του Hello World
Για να ελέγξετε ότι όλα λειτουργούν κανονικά, εκτελέστε την εντολή:
```shell
docker run --name my-container hello-world
```

Τι κάνει αυτή η εντολή:
1.  Αν δεν υπάρχει ήδη το image `hello-world` στο τοπικό σας σύστημα, το Docker θα το **κατεβάσει** αυτόματα από το **DockerHub**.
2. Θα εκτελέσει το image μέσα σε ένα **container**.
3. Θα εμφανιστεί στην οθόνη σας ένα μήνυμα επιτυχίας, παρόμοιο με το εξής:
```
Hello from Docker!
```

# **✅ Συμπεράσματα**
Αν εμφανίστηκε το μήνυμα **“Hello from Docker!**”, τότε η εγκατάσταση του Docker σας είναι επιτυχής και είστε έτοιμοι να δημιουργήσετε και να εκτελέσετε τα δικά σας containers!

## Προαιρετικό: Διαγραφή του Image
Αν θέλετε να διαγράψετε το image για να ελευθερώσετε χώρο:
```shell
# Διαγράφει το Container
docker rm my-container

# Διαγράφει το Image
docker rmi hello-world
```
